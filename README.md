# MASTER V10: بنية أتمتة وهيكلة التحليل القانوني الجنائي
    
## ⚖️ وصف المنظومة
نظام متطور يعتمد على هندسة الاستدعاء المعزز بالجرّ (RAG) والذكاء الاصطناعي السيادي لتفكيك، وصهر، وتبويب الأحكام القضائية الصادرة عن محكمة النقض المصرية وتوزيع القواعد المستخلصة منها آلياً على شجرة مجلدات الجنايات الـ 21.

## 📂 الهيكل التنظيمي لشجرة المستودع
- **knowledge / jurisprudence**: المجلد المركزي الحاضن لقواعد البيانات المعرفية المصهورة (ملفات JSON).
- **المجلدات الفرعية الـ 21 المستهدفة**:
  - `judgment_reasoning` (تسبيب الأحكام)
  - `narcotics_control` (مكافحة المخدرات)
  - `public_funds_embezzlement` (اختلاس الأموال العامة)
  - `sexual_assault_crimes` (جرائم الاعتداء الجنسي)
  - *وبقية الفروع التخصصية الجنائية المستقرة صلب النظام.*

## 🚀 بروتوكول التشغيل والأتمتة
يتم سحب المستندات بصيغة `DOCX` من الجسر السحابي للـ Google Drive، وصهرها وتوليد الـ JSON النقي عبر معالجات Gemini، ثم المزامنة التلقائية والرفع الآمن مباشرة إلى هذا المستودع الرقمي.
